// LINHA DE COMANDO (CLI) - https://ss64.com/ ------MOVIMENTAR (BASH) $ cd sites - Move para a pasta sites $ cd .. - Move para a pasta anterior $ cd ~/Desktop/sites - Move para a pasta sites dentro de Desktop $ cd ~ - Move para o diretório principal do sistema / home $ clear - Limpa a tela ------LISTAR E CRIAR (BASH) $ ls - Lista diretórios e arquivos $ ls -all - Lista diretórios, arquivos, invisíveis e detalhes $ mkdir site - Cria o diretório site $ touch index.html - Cria o arquivo index.html ------REMOVER (BASH) $ rm index.html - Remove o arquivo index.html $ rm -r site - Remove o diretório site e todos os arquivos dentro dele seta para cima / baixo - Acessa o comando anterior tab - Auto-completa o comando
